# SDGP-CareerGuide — GrowthMap
Career guidance platform that gives local qualifications and local institutes as suggestions for Sri Lankan ICT students.

## ✨ New Features (v2)
- 🌙 **Dark Mode Toggle** — Switch between light/dark from the Navbar (persisted in localStorage)
- 📊 **Animated Stats Counter** — Home page section with animated numbers (students guided, careers mapped, etc.)
- 🔖 **Career Bookmarks** — Save/unsave careers with a Saved filter tab on the Careers page
- ⏱️ **Quiz Timer** — 30-second countdown per question with auto-skip and visual progress bar
- 🏆 **Assessment History** — Past quiz results displayed on the Profile page

## Tech Stack
- React + Vite + Tailwind CSS
- Framer Motion for animations
- localStorage for persistence
- Python/Flask backend (ML prediction)
